# Social Media & Mental Health Balance
## Descrição

Este projeto tem como objetivo analisar a relação entre o uso de redes sociais e o equilíbrio da saúde mental.
Através de um conjunto de dados com 500 registros, o estudo busca identificar padrões entre tempo de tela, níveis de estresse, qualidade do sono e felicidade.

O projeto foi desenvolvido como parte do aprendizado em análise de dados com Python, utilizando bibliotecas como pandas, matplotlib e seaborn.

## Objetivos
- Explorar e limpar um conjunto de dados sobre comportamento digital.
- Investigar correlações entre tempo de tela e bem-estar mental.
- Gerar visualizações simples e intuitivas.
